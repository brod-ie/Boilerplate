# Brodie Boilerplate
A boilerplate implementing my tech stack of choice.

## Quick start

1. Fork this repo
2. `$ git clone https://github.com/your-username/Boilerplate.git`
3. `$ cd path/to/Boilerplate && npm run build`

## Running the server

1. `$ npm start`
2. Navigate to [`http://localhost:3000`](http://localhost:3000)
